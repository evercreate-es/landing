# Industry Variables for Cold Email

Each row provides the variables needed to customize the 3 email templates per industry.

## Variable Reference

| Variable | Used in | Description |
|----------|---------|-------------|
| `pain_points` | Email 1 | What scattered tools/processes they're replacing |
| `case_study_line` | Email 2 | One-sentence result from a similar company |
| `humor_variant` | Email 3 | Industry-adapted joke for option 3 |
| `code` | Email 2 | Promo code for landing page |
| `industry_slug` | Email 2 | URL path (evercreate.co/{{slug}}) |

---

## Default Humor Variant

Used for all industries unless overridden below:

> Quick! Call 911! I fell in the shower and broke my arm!
